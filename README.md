# tradeoffs-claims

 Claims Decision Support App

Interactive tool for evaluating claim risk and filing tradeoffs.

## What it does
- Analyzes claim timing vs. denial risk
- Helps identify which claims to prioritize or exclude
- Surfaces tradeoffs across a batch
